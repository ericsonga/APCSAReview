How to solve this problem¶

The first thing to do is to create a new instance of the ClimbInfo object, which will be added to the climbList array.

    X-X-1: Which of the following correctly creates a new ClimbInfo object?
  • ClimbInfo climb = ClimbInfo(peakName, climbTime);
  • The keyword 'new' is missing. This is required to create a new instance of a class.
  • ClimbInfo climb = new ClimbInfo(climbTime, peakName);
  • Double check the order of the parameters in the constructor.
  • ClimbInfo climb = new ClimbInfo(peakName, climbTime);
  • This will correctly create a new ClimbInfo object.

Next, we need to know how to add to the end of a list.

    X-X-2: Which of the following correctly adds the element infoObj to the end of the ‘climbList’ list?
  • climbList.add(infoObj);
  • The add method with one parameter appends the element to the end of the list.
  • climbList[nextIndex] = infoObj;
  • This is the syntax for setting the value of an array at an index.
  • climbList.add(0, infoObj);
  • This will add infoObj at index 0, but first move to the right any existing items.

Try to write the code for the method addClimb. When you are ready click “Run” to test your solution.

import java.util.List;
import java.util.ArrayList;

public class ClimbingClub
{
   /** The list of climbs completed by members of the club.
    *  Guaranteed not to be null. Contains only non-null references.

   public ClimbingClub()
   {
      climbList = new ArrayList();
   }

   /** Adds a new climb with name peakName and time climbTime to
    *  the end of the list of climbs
    *
    * @param peakName the name of the mountain peak climbed
    * @param climbTime the number of minutes taken to complete the climb
    */
   public void addClimb(String peakName, int climbTime)
   {
     /* to be implemented in part a */
   }

   public String toString()
   {
     String output ="";
     for (ClimbInfo info : climbList)
     {
       output = output + info.toString() + "\n";
     }
     return output;
   }
